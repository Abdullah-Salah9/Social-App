# Social App / Linked Posts

A React.js-based social platform where users can post, comment, and upload images.  
The app demonstrates front-end authentication, global state management with Context API, and responsive design using Tailwind CSS.

## 🔹 Features
- User registration and login (front-end authentication)
- Create, edit, and delete posts
- Add and delete comments
- Upload images with posts
- Global state management with **Context API**
- Routing across pages: Login, Home (Feed), Profile
- Fully responsive layout with **Tailwind CSS**

## 🔹 Technologies Used
- **React.js** (Front-End Library)
- **Context API** (Global state management)
- **Tailwind CSS** (Responsive styling)
- **RESTful APIs** (Backend integration for posts, users, and comments)
- **Local Storage** (Temporary storage if API offline)
